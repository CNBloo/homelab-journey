# Day 1 - Building My First Homelab
**Date:** July 20, 2026

## Objective
Begin building my first homelab using Proxmox and start learning virtualization, Linux, and self-hosting.

## Hardware
- HP ProDesk 600 G1
- Intel Core i5-4570
- Single HDD (initial setup)

## What I Accomplished
- Installed Proxmox VE
- Learned how to create bootable USBs using Rufus
- Created my first virtual machines
- Installed Jellyfin
- Explored Jellyfin plugins
- Learned the basics of storage management and disk formatting
- Spent time troubleshooting installation and boot issues

## Challenges
- Troubleshooting Windows installation
- Creating a bootable USB correctly
- Learning the Proxmox interface
- Understanding storage configuration and formatting
- Setting up my first virtual machines

## What I Learned
- How to use Rufus to create bootable USB drives
- Basic Proxmox administration
- Creating and managing virtual machines
- How Jellyfin organizes media libraries
- How Jellyfin plugins extend functionality
- Basic storage and disk management concepts

## Lessons Learned
Today reminded me that building a homelab is more about learning than getting everything perfect on the first try. Most of the day was spent troubleshooting, but each issue helped me better understand how Proxmox, storage, and virtualization work.
